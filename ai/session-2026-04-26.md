## Quyết định kỹ thuật đáng nhớ

### React Query KHÔNG dùng cho login
`useMutation` từ `@tanstack/react-query` **không phù hợp** cho Firebase Auth login vì:
- React Query quản lý **server state** (cache, sync, refetch) — login không có server state
- Firebase Auth tự quản lý session riêng
- `useState` + `async/await` là pattern đúng và gọn hơn cho use case này

React Query **nên dùng** trong auth module khi: fetch user profile từ Firestore, update profile, check token validity.
